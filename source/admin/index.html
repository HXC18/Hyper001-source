HOW IT WORKS

这套线上后台怎么工作

  • `/admin/` 页面由 Hexo 直接生成，继续托管在 GitHub Pages
  • 后台读写请求发给 Cloudflare Worker，不暴露 GitHub Token 到浏览器
  • Worker 通过 GitHub API 修改 `source/_posts` 和 `_config.fluid.yml`
  • 仓库更新后，GitHub Actions 自动执行 `npm run build` 和 `npm run deploy`
